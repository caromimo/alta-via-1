# Italian shelters on the Alta Via 1

Based on the [API](https://rifugi.cai.it/api/documentation#/).

The AV1 gpx file is from https://walkingenglishman.com/europeldp/03altavia1.html
Which was converter to a geojson format using https://geojsonconverter.vercel.app/
Which was visualised here https://geojson.io/

The initial dataset is from the [Italian Alpine Club](https://www.cai.it/), aka in italian as the CAI or the Club Alpino Italiano.
